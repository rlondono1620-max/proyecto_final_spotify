Proyecto Final - Dashboard Spotify y YouTube

Este proyecto presenta un análisis de datos entre las métricas de YouTube (vistas, likes, comentarios) y los streams en Spotify.

Contiene:
- app.py → código del dashboard
- Spotify_Youtube.csv → dataset de análisis
- requirements.txt → librerías necesarias

Para ejecutar:
1. Instala las dependencias:
   pip install -r requirements.txt
2. Ejecuta el dashboard:
   python app.py
3. Abre en el navegador:
   http://127.0.0.1:8050
